# 🌍 World Happiness Report 2024 Analysis

## 📖 Project Overview
This project provides a comprehensive analysis of the **2024 World Happiness Report**. By integrating **SQL**, **Python**, and **Power BI**, I explored how different socio-economic factors like GDP, Social Support, and Freedom influence global happiness levels.

---

## 🖥️ Interactive Power BI Dashboard
The Power BI dashboard features interactive elements that allow users to filter data by country and compare metrics against global benchmarks.

### Key Visuals:
*   **KPI Cards:** Track "Social Support" and "GDP per capita" against world averages.
*   **Top 10 Rankings:** A bar chart showing the happiest nations.
*   **Q&A AI Visual:** Natural language search for country-specific data.

## 📈 Key Insights & Metrics
*   **Economic Strength:** GDP remains a primary driver, but not the only one. 
*   **Social Support:** Countries with high Social Support targets ( > 0.78) consistently rank in the top 20.
*   **Data Reliability:** By comparing **Upper** and **Lower Whiskers**, we identified the most stable data points in the 2024 report.
